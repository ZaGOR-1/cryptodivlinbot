# Условия использования — cryptodivlinbot

_Последнее обновление: 2026-04-26_

Используя Telegram-бот **cryptodivlinbot**, вы соглашаетесь с
условиями ниже. Бот с открытым кодом:
<https://github.com/ZaGOR-1/cryptodivlinbot>. Развёрнутые инстансы
эксплуатируются отдельными третьими сторонами (операторами). Английская
версия в `TERMS_OF_SERVICE.md` — основная; русская приведена для
удобства.

## 1. Что делает бот

cryptodivlinbot отслеживает топ-N криптовалют по капитализации и
присылает:

- **Периодический дайджест** текущих цен и изменений за 24 часа.
- **Spike-алерты** при движении монеты сильнее заданного процента
  за заданное время.

Подпиской управляете через `/subscribe` / `/unsubscribe`,
чувствительностью алертов — через `/setthreshold`.

## 2. Не является финансовым советом

**Информация, которую отдаёт бот — справочная.** Это **не** является
и не должно восприниматься как финансовый, инвестиционный, торговый,
налоговый или юридический совет. Криптовалюты крайне волатильны; вы
можете потерять весь капитал. Проводите собственное исследование и
консультируйтесь с лицензированным советником перед принятием
решений.

Оператор и контрибьюторы исходного кода **не гарантируют** точности,
полноты, актуальности или пригодности для конкретной цели данных,
которые присылает бот. Источники (CoinGecko, Binance) могут отдавать
неточные, задержанные или отсутствующие цены.

## 3. Допустимое использование

Запрещается:

- Использовать бота для манипуляций, обмана или мошенничества.
- Реверс-инжинирить или атаковать инфраструктуру бота вне рамок
  лицензии исходного кода.
- Спамить бота абьюзивным объёмом запросов, нарушающим сервис для
  остальных.

Оператор может ограничить или отключить чаты, явно нарушающие этот
пункт.

## 4. Конфиденциальность

Как бот работает с вашими данными — описано в
[Политике конфиденциальности](./PRIVACY_POLICY_RU.md). Кратко: бот
хранит только chat id, язык, порог, состояние подписки и
кратковременные кулдауны. Полное удаление — `/forgetme`.

## 5. Доступность и изменения

Бот предоставляется **как есть**, по **best-effort**. Оператор может
остановить его, изменить функционал или прекратить работу в любой
момент без уведомления.

и авторы исходного кода **не несут ответственности** за прямые,

бизнесе — связанные с использованием или невозможностью использования
бота.

## 7. Сторонние источники данных

Рыночные данные — из публичных API CoinGecko (основное) и Binance
(резервное). Пользование этими API регулируется их собственными
условиями.
